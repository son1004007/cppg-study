# CPPG 암기 전략

## 목적
문제풀이를 반복하면서 개념을 암기하고, 실제 시험에서 선택지를 구분할 수 있는 수준까지 정확도를 높입니다.

## 핵심 원칙
- 이론 정독보다 문제 반복을 우선합니다.
- 틀린 문제를 중심으로 재회독합니다.
- 왜 틀렸는지 원인을 분류합니다.
- 헷갈리는 개념은 비교표로 정리합니다.

## 오답 원인 분류
1. 개념 자체를 모름
2. 용어가 헷갈림
3. 예외 조항을 기억하지 못함
4. 지문 함정에 걸림
5. 정답은 아는데 선택지 비교에서 흔들림

## 반복 방식
### 1차
- 전체 문제를 빠르게 풉니다.
- 모르는 문제는 바로 해설을 보고 핵심 키워드만 남깁니다.

### 2차
- 틀린 문제만 다시 풉니다.
- 왜 틀렸는지 원인을 적습니다.

### 3차
- 자주 틀리는 주제를 따로 묶어서 반복합니다.
- 비교표를 만들어 유사 개념을 정리합니다.

## 비교표 권장 주제
- 개인정보 / 가명정보 / 익명정보
- 개인정보처리자 / 수탁자
- 동의 / 동의 예외
- 목적 외 이용 / 제3자 제공
- 민감정보 / 고유식별정보
- 정보주체 권리 / 개인정보처리자의 의무

## 실전 운영
- 하루 공부 시간이 짧아도 회독을 끊지 않습니다.
- 새 문제를 늘리기보다 틀린 문제를 줄이는 데 집중합니다.
- 정답 문장 전체를 외우기보다 구분 기준을 외웁니다.
