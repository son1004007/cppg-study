# 다음 시험 대비 전략

## 목적
이번 CPPG 시험 회고를 바탕으로,
다음 유사 자격증 시험에서는 같은 실수를 반복하지 않도록 준비 전략과 시험장 운영 기준을 정리한다.

## 핵심 결론
다음 시험은 단순히 더 많이 외우는 방식으로 준비하면 안 된다.
핵심은 아래 두 가지를 동시에 훈련하는 것이다.

- 정답 기준 암기
- 제한 시간 안에 푸는 실전 운영 능력

## 이번 시험에서 확인된 핵심 문제
- 45번을 풀고 있을 때 60분이 지나 있었다.
- 마지막 5문제는 지문도 다 읽지 못하고 찍었다.
- 답 마킹을 5분 남기고 시작했다.
- 보기 2개 중에서 정확히 알지 못하는 문제를 너무 오래 잡고 있었다.

## 원인 정리
### 1. 지식 부족만의 문제가 아니었음
암기 자체가 전혀 안 된 것이 아니라,
시험장에서 시간을 통제하지 못하면서 후반부가 무너졌다.

### 2. 애매한 문제 손절 실패
2지선다 문제를 오래 붙잡으면서 전체 시간 배분이 무너졌다.

### 3. 마킹 전략 부재
마킹 시간을 별도로 확보하지 않아 후반부 압박이 커졌다.

## 다음 시험 준비의 기본 원칙
### 1. 아는 문제를 끝까지 가져간다
모르는 문제를 맞히려고 전체 시간을 쓰지 않는다.
아는 문제를 빠르게 처리해서 확실한 점수를 확보하는 것이 우선이다.

### 2. 애매한 문제는 오래 잡지 않는다
2개 보기 중에서 확신이 없으면 표시만 하고 넘긴다.
시험은 한 문제를 완벽히 푸는 게임이 아니라 전체 점수를 최대화하는 게임이다.

### 3. 시간은 지식만큼 중요하다
다음 시험에서는 정확도 훈련과 함께 속도 훈련을 반드시 병행해야 한다.

### 4. 마킹 시간은 별도로 확보한다
마킹은 마지막 5분에 몰아서 하지 않는다.
최소 10~15분 전에는 마킹이 끝난 상태를 목표로 한다.

## 다음 시험 공부 방식
### 유지할 것
- 문제집 중심 회독
- 전자책 읽어주기 기능 활용
- Q/A 암기형 정리
- 틀린 이유 기록
- 비교표 정리
- 숫자 / 예외 / 주체 / 순서 중심 압축 암기

### 바꿀 것
- 애매한 문제를 오래 붙잡는 습관
- 해설을 길게 읽는 방식
- 실전 시간 측정 없이 회독만 반복하는 방식
- 시험 종료 직전까지 문제만 풀고 마킹을 미루는 방식

## 학습 비중 권장
다음 시험 준비에서는 아래 비중으로 학습한다.

- 60%: 문제풀이 회독
- 20%: 오답 / 비교표 / 약점 정리
- 20%: 시간 재고 푸는 실전형 훈련

## 실전 훈련 방법
### 1. 2지선다 손절 훈련
다음 규칙을 고정한다.

- 20~30초 안에 판단 안 되면 표시하고 넘긴다.
- 다시 돌아와도 1분 이상 쓰지 않는다.
- 끝까지 모르겠는 문제는 과감히 버린다.

### 2. 시간 체크 포인트 설정
시험 시작 전 시간 기준을 미리 정한다.

예시:
- 30분 시점: 최소 특정 번호 도달
- 60분 시점: 최소 특정 번호 도달
- 마지막 15분: 마킹 완료 상태

핵심은 60분 시점에 45번 수준이면 늦다는 점을 명확히 인지하는 것이다.

### 3. 마킹 포함 시뮬레이션
실전 연습 시 문제만 푸는 것이 아니라,
마킹까지 포함해서 연습한다.

### 4. 회독 구조 분리
시험장에서는 아래처럼 운영한다.

- 1회독: 바로 아는 문제만 처리
- 2회독: 표시한 문제 다시 보기
- 3회독: 남은 문제 중 확률 높은 것만 판단

## 다음 시험 준비 기간별 전략
### 시험 4주 전
- 전 범위 1회독
- 개념별 비교표 정리 시작
- 숫자 / 예외 / 주체 / 순서 문제 분류

### 시험 3주 전
- 문제풀이 회독 반복
- 오답노트 축적
- Q/A 암기형 정리 확장

### 시험 2주 전
- 시간 재고 문제풀기 시작
- 2지선다 손절 훈련
- 마킹 포함 시뮬레이션

### 시험 1주 전
- 새 범위 확장 금지
- 오답 / 비교표 / 숫자 / 예외 / 순서만 반복
- 실전 속도 고정

### 시험 전날
- 전체 범위 다시 보기 금지
- 압축 노트만 보기
- 컨디션 관리

## 시험장 운영 원칙
- 아는 문제 먼저 푼다.
- 애매하면 표시하고 넘긴다.
- 모르는 문제를 오래 보지 않는다.
- 마지막 10~15분은 반드시 확보한다.
- 마지막 5문제는 최소한 읽고 판단할 시간이 남도록 운영한다.

## 다음 시험에서 꼭 기억할 한 줄
- 정확도만으로는 합격이 안 될 수 있다.
- 시간 통제도 실력이다.
- 2지선다에서 오래 고민하지 않는 것이 점수를 지키는 방법이다.

## 최종 결론
다음 시험은 암기력을 조금 더 높이는 수준으로 접근하면 안 된다.
정답 기준 암기, 문제 풀이 속도, 손절 기준, 마킹 시점까지 포함한 실전형 전략으로 준비해야 한다.
