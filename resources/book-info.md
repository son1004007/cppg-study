# 교재 정보

## 사용 교재

| 항목 | 내용 |
|---|---|
| 교재명 | 2026 이기적 CPPG 개인정보관리사 기출 900제 |
| 형태 | eBook / PDF |
| 구매처 | YES24 |
| 핵심 특징 | 기출 900제, 문제풀이 중심, 해설 기반 학습 가능 |

## 교재 활용 방식

```text
문제 풀기 → 해설 보기 → 다시 문제 풀기
```

- YES24 전자책으로 구매하여 사용합니다.
- 문제를 직접 풀고, 정답과 해설을 확인하면서 핵심 내용을 암기합니다.
- 해설 확인 후 같은 문제 또는 유사 문제를 다시 풀어 선택지 판단 기준을 익힙니다.
- 읽어주기 기능을 이용해 문제와 보기를 반복 청취합니다.
- 틀린 문제와 헷갈리는 개념은 별도로 누적 정리합니다.

## 이 교재를 사용하는 이유

- 기출 문제 중심이라 시험 표현에 익숙해지기 좋습니다.
- 회독 수를 빠르게 늘리기 좋습니다.
- 해설을 통해 필요한 이론만 역으로 학습하기 좋습니다.
- 읽어주기 기능과 결합하면 자투리 시간 활용이 쉽습니다.

## 주의할 점

- 문제와 답만 외우지 않도록 해설의 기준 문장을 같이 확인합니다.
- 반복해서 틀리는 개념은 비교표와 오답노트로 보완합니다.
- 정답 번호가 아니라 왜 그 선택지가 정답인지 설명할 수 있어야 합니다.
