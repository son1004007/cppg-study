## 사용 방식

- YES24 eBook으로 구매하여 사용한다.
- 스마트한 PDF 필기 기능을 활용할 수 있다.
- 문제를 먼저 풀고, 해설을 본 뒤 다시 문제를 푸는 방식으로 회독한다.
- 읽어주기 기능을 활용해 이동 시간에도 문제와 보기를 반복 청취한다.

## 핵심 공부 방식

```text
문제 풀이 → 해설 확인 → 다시 문제 풀이
```
