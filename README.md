다음에도 비슷한 자격증 시험에 재사용할 수 있도록 정리한 저장소입니다.

- 시험: CPPG 개인정보관리사
- 시험일: 2026-04-12
- 취득일: 2026-04-29
- 결과: 합격
- 유효기간: 2028-04-29까지

## 빠른 요약
- 적용 시험: CPPG

- 재사용 목적: 이후 객관식·기출 중심 자격증 시험에도 같은 방식 적용

## 이번에 사용한 공부 방식

### 1. 문제를 먼저 푼다
- 이론 정독보다 문제집 회독을 우선했습니다.
- 처음부터 완벽히 이해하려고 하지 않고, 실제 출제 표현과 선택지에 먼저 익숙해졌습니다.

### 2. 해설을 본다
- 문제를 푼 뒤 정답과 해설을 확인했습니다.
- 해설에서 정답 근거, 오답 선택지의 함정, 법령상 예외를 확인했습니다.

### 3. 다시 문제를 푼다
- 해설을 본 뒤 같은 문제 또는 유사 문제를 다시 풀었습니다.
- 단순히 정답 번호를 외우는 것이 아니라, 선택지를 구분하는 기준을 익혔습니다.

### 4. 읽어주기 기능을 활용한다
- YES24 전자책 읽어주기 기능으로 문제와 보기를 반복 청취했습니다.
- 눈으로 공부하기 어려운 시간에도 회독을 이어가려는 목적입니다.

### 5. 시험 직전에는 압축 회독한다
- 전체 범위를 다시 보지 않고, 오답·약점·비교표 중심으로 압축 회독했습니다.

## 이 방식을 다시 적용할 기준
아래 조건이면 같은 방식을 우선 적용합니다.

- 객관식 시험이다.
- 기출 문제집이 있다.
- 단기 합격이 목표다.
- 암기와 선택지 구분이 중요하다.
- 자투리 시간을 활용해야 한다.
