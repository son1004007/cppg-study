# CPPG Study

## 목표
- CPPG 자격증 합격
- 이론 정독보다 문제풀이 중심 회독으로 합격 가능한 수준의 암기 완성

## 현재 학습 방식
- YES24 전자책으로 문제집 위주 교재를 사용합니다.
- 읽어주기 기능을 활용해 문제와 보기를 반복 청취합니다.
- 문제를 직접 풀면서 정답과 해설을 확인합니다.
- 틀린 문제와 헷갈리는 개념을 따로 정리합니다.
- 문제를 반복해서 풀며 내용 자체를 외우는 방식으로 학습합니다.

## 이 방법을 선택한 이유
- 긴 이론 정독보다 문제 중심 학습이 집중 유지에 더 유리합니다.
- 읽어주기 기능을 이용하면 눈이 피곤한 상태에서도 학습을 이어갈 수 있습니다.
- 문제와 선택지 표현을 반복해서 접하면서 시험형 암기를 강화할 수 있습니다.

## 학습 절차
### 1. 문제 먼저 풀기
- 문제를 먼저 읽고 답을 추론합니다.
- 모르면 바로 정답과 해설을 확인합니다.

### 2. 읽어주기 기능으로 반복 청취
- YES24 읽어주기 기능으로 문제와 보기를 반복해서 듣습니다.
- 출퇴근, 이동, 피로한 시간에도 청취 중심으로 회독합니다.

### 3. 오답 누적
- 틀린 문제는 다시 봅니다.
- 자주 헷갈리는 개념은 별도 메모로 축적합니다.

### 4. 약점 위주 재회독
- 전체를 반복하기보다 틀린 문제와 헷갈리는 주제를 우선 재회독합니다.
- 비슷한 개념은 비교해서 외웁니다.

## 현재 방식의 장점
- 회독 수를 빠르게 늘릴 수 있습니다.
- 시험 문제 표현에 익숙해질 수 있습니다.
- 자투리 시간을 활용하기 좋습니다.

## 현재 방식의 한계
- 문제와 답만 외우면 응용형 문항에 약할 수 있습니다.
- 개념 체계가 약하면 비슷한 선택지에서 흔들릴 수 있습니다.

## 보완 방법
- 반복해서 틀리는 주제는 3~5줄 개념 요약으로 정리합니다.
- 단순 정답 암기가 아니라 왜 정답인지 한 줄 설명을 붙입니다.
- 아래 주제는 비교표로 정리합니다.
  - 개인정보 / 가명정보 / 익명정보
  - 개인정보처리자 / 수탁자
  - 동의 필요 / 동의 예외
  - 목적 외 이용 / 제3자 제공

## 운영 원칙
- 매일 조금이라도 끊기지 않고 반복합니다.
- 새 문제를 많이 푸는 것보다, 틀린 문제를 다시 맞히는 데 집중합니다.
- 완벽한 이해보다 시험 합격에 필요한 정확한 암기를 우선합니다.
