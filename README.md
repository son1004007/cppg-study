# cppg-study

## 개요
이 레포는 2026-04-12 CPPG 시험 준비 과정에서 사용한 공부 방법을 기록하고,
다음에도 비슷한 자격증 시험에 재사용할 수 있도록 정리한 저장소입니다.

## 빠른 요약
- 적용 시험: CPPG
- 시험일: 2026-04-12
- 사용 교재: 2026 이기적 CPPG 개인정보관리사 기출 900제
- 핵심 방식: 문제집 회독 + YES24 읽어주기 기능 + 오답/약점 압축 회독
- 재사용 목적: 이후 객관식·기출 중심 자격증 시험에도 같은 방식 적용

## 이 레포에서 먼저 볼 문서
1. `study-method-summary.md`
   - 다른 채팅이나 다음 시험 준비 시 가장 먼저 보면 되는 요약 문서
2. `strategy/reusable-exam-study-method.md`
   - 재사용 가능한 자격증 공부 방법 상세 정리
3. `resources/book-info.md`
   - 이번 시험에서 사용한 교재 정보
4. `study-log/weak-points.md`
   - 헷갈리는 개념과 약점 정리
5. `templates/daily-log-template.md`
   - 다음 시험에도 바로 쓸 수 있는 일일 학습 기록 템플릿

## 이번에 사용한 공부 방식
### 1. 문제집 중심으로 시작
- 이론 정독보다 문제집 회독을 우선했습니다.
- 문제를 먼저 풀고, 모르는 내용만 해설로 확인했습니다.

### 2. 읽어주기 기능 활용
- YES24 전자책 읽어주기 기능으로 문제와 보기를 반복 청취했습니다.
- 눈으로 공부하기 어려운 시간에도 회독을 이어가려는 목적입니다.

### 3. 오답 정리 방식
- 틀린 문제 번호보다 왜 틀렸는지를 정리했습니다.
- 개념 부족, 용어 혼동, 예외 규정 미암기, 선택지 구분 실패로 분류하는 방식입니다.

### 4. 시험 직전 압축 회독
- 전체 범위를 다시 보지 않고, 오답·약점·비교표 중심으로 압축 회독했습니다.

## 이 방식을 다시 적용할 기준
아래 조건이면 같은 방식을 우선 적용합니다.

- 객관식 시험이다.
- 기출 문제집이 있다.
- 단기 합격이 목표다.
- 암기와 선택지 구분이 중요하다.
- 자투리 시간을 활용해야 한다.
