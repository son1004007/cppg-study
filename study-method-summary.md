# 자격증 시험 공부 방법 요약

## 이 문서의 목적
이 문서는 CPPG 시험 준비 과정에서 실제로 사용한 공부 방식을 정리한 요약본입니다.
다른 채팅이나 이후 유사한 자격증 시험 준비 시에도 바로 참고할 수 있도록 작성합니다.

## 적용한 시험
- CPPG 개인정보관리사
- 시험일: 2026-04-12
- 취득일: 2026-04-29
- 결과: 합격
- 사용 교재: 2026 이기적 CPPG 개인정보관리사 기출 900제

## 핵심 결론
이번에 사용한 방식은 다음과 같습니다.

```text
문제 풀기 → 해설 보기 → 다시 문제 풀기
```

세부 방식은 다음과 같습니다.

- 이론 정독보다 문제집 회독을 우선합니다.
- 문제를 먼저 풀고, 해설을 통해 필요한 개념을 역으로 익힙니다.
- 해설 확인 후 같은 문제 또는 유사 문제를 다시 풉니다.
- 틀린 문제 전체보다 틀린 이유와 헷갈리는 개념을 따로 누적 정리합니다.
- YES24 전자책 읽어주기 기능으로 문제와 보기를 반복 청취합니다.
- 시험 직전에는 전체 범위보다 오답, 비교표, 약점만 압축 회독합니다.

## 왜 이 방식이 맞았는가
- 객관식 시험은 문제 표현과 선택지 패턴에 익숙해지는 것이 중요합니다.
- 단기 합격에서는 완벽한 이론 정독보다 출제 포인트를 빠르게 잡는 것이 효율적입니다.
- 해설을 본 뒤 다시 풀어야 정답 번호 암기가 아니라 판단 기준이 남습니다.
- 자투리 시간에도 회독을 이어갈 수 있어야 합니다.
- 읽어주기 기능을 활용하면 눈으로만 공부할 때보다 반복 노출량을 늘릴 수 있습니다.

## 실제 공부 방식

### 1. 문제를 먼저 푼다
- 이론서를 처음부터 끝까지 읽지 않습니다.
- 기출 문제를 먼저 풀면서 어떤 방식으로 묻는지 확인합니다.
- 모르는 개념이 나와도 우선 문제를 끝까지 풉니다.

### 2. 해설을 본다
- 정답 근거를 확인합니다.
- 오답 선택지가 왜 틀렸는지 확인합니다.
- 숫자, 예외, 주체, 절차처럼 시험에 자주 나오는 기준을 표시합니다.

### 3. 다시 문제를 푼다
- 해설을 본 뒤 다시 문제를 풉니다.
- 같은 문제를 다시 풀 때는 정답 번호가 아니라 근거를 말할 수 있어야 합니다.
- 유사 선택지에서 헷갈리면 약점 노트에 남깁니다.

### 4. 읽어주기 기능 활용
- YES24 전자책 읽어주기 기능으로 문제와 보기를 반복해서 듣습니다.
- 출퇴근, 이동, 피로한 시간에도 회독을 이어갑니다.

### 5. 오답 정리 방식
- 틀린 문제 번호 자체보다 왜 틀렸는지를 기록합니다.
- 아래 기준으로 분류합니다.
  - 개념을 모름
  - 용어가 헷갈림
  - 예외 규정을 기억하지 못함
  - 비슷한 선택지를 구분하지 못함
  - 지문 함정에 걸림

### 6. 약점 압축 회독
- 전체를 계속 반복하기보다 자주 틀리는 주제만 다시 봅니다.
- 유사 개념은 비교표로 정리합니다.

## 다음 시험에도 그대로 적용할 기준
아래 조건이면 같은 방식을 우선 적용합니다.

- 객관식 시험이다.
- 기출 문제집이 있다.
- 단기 합격이 목표다.
- 암기와 선택지 구분이 중요하다.
- 자투리 시간을 활용해야 한다.

## 다음 시험에서 조정할 기준
- 법령형 시험이면 숫자, 예외 규정, 주체 구분을 더 따로 정리합니다.
- 계산형 시험이면 오답노트보다 풀이 패턴 정리를 강화합니다.
- 서술형 시험이면 문제풀이 중심 방식만으로는 부족하므로 개념 요약본을 추가합니다.

## 한 줄 정리
문제를 풀고, 해설을 보고, 다시 문제를 푸는 방식으로 기출 회독을 반복하면 객관식·기출 중심 시험에서 합격 점수를 만들 수 있습니다.
